## 🚀 Quick Start

### 1. Extract the Archive
```bash
tar -xzf econ_discord_py.tar.gz
cd econ_discord_py
```

### 2. Install Dependencies
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

```

### 3. Configure
```bash
cp .env.example .env
# Edit .env with your Discord token and database settings
```

### 4. Run
```bash

```

## 🔑 Key Differences from JavaScript

| Feature | Discord.js | Discord.py |
|---------|-----------|-----------|
| **Database** | Sequelize | SQLAlchemy |
| **Commands** | Builder pattern | Decorator pattern |
| **Async** | Promises/async-await | async-await |
| **Modules** | require() | import/from |
| **Intents** | GatewayIntentBits | discord.Intents |
| **Embeds** | EmbedBuilder | discord.Embed |

### Environment Variables

**Required:**
- `TOKEN` - Your Discord bot token

**Database (choose one):**
- PostgreSQL: `DB_TYPE=postgresql` + connection vars
- SQLite: `DB_TYPE=sqlite` + `DB_PATH`

**Optional:**
- `SERVER_BYPASS` - Comma-separated server IDs for premium bypass
- `SQL_ECHO` - Set to `true` to see database queries

## 🐛 Troubleshooting

### Bot won't start
- Check Discord token in `.env`
- Verify Python 3.10+ installed: `python --version`
- Check all dependencies: `pip list | grep discord`

### Database connection fails
- PostgreSQL: Ensure database exists: `createdb econ_discord`
- SQLite: Check file path in `.env`
- Try `DB_TYPE=sqlite` for quick testing

### Commands not showing up
- Run `/sync` (if implemented)
- Check cog loading in console output
- Verify bot has proper permissions in Discord
